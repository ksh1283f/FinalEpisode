전투 관련 요소

<UI>
- 스킬 자원바(공격, 유틸, 방어)
- 유저스펙(무기, 방어구, 장신구, 체력바, 버프, 스텟)
- 적 패턴관련(패턴 아이콘, 스킬, 캐스팅 바, 체력바)
- 페이즈
- 스킬버튼(필러, 사용)

<시스템>
- 유저스펙 갱신(전투 시작 전 제일먼저 작동해야함)
- 적 패턴, 난이도 적용(데이터 파일을 불러와서-> 데이터 파일 형식 구상하기)
- 필러스킬 버튼 작동
- 사용스킬 버튼 작동(메커니즘 적용: 사용시 모든 자원이 사용)
- 버튼사용에 따른 상호작용(스킬 발사, 필러스킬 사용 시 자원 생성)
- 전투가 끝난 경우에 따른 처리

<던전 패턴 데이터 형식: DungeonPattern>
- 패턴 id: int
- 패턴 이름: string
- 적 체력: int
- 적 패턴리스트: List<EnemyPattern>
 : EnemyPattern
 -> 대응해야되는 스킬 종류: enum
 -> 패턴 캐스팅 시간: float
 -> 패턴 설명(필요한 경우에만 출력): string
- 패턴 텀(유저의 대응에 따라 바뀔수 있는 요소): float

--> 패턴 데이터는 csv형식으로 작성
